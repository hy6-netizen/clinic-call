최초 1회만 설정하면 됩니다. 두 기기 모두 같은 값으로 입력해주세요.

1
Firebase 설정값

Firebase 콘솔에서 복사한 firebaseConfig 전체를 아래에 붙여넣으세요.

Firebase 설정 방법 안내 ▾
  1. console.firebase.google.com 접속 (구글 계정 필요)
  2. "프로젝트 추가" → 이름 입력 (예: clinic-call) → Analytics는 끄셔도 됩니다
  3. 왼쪽 메뉴 "빌드" → "Realtime Database" → "데이터베이스 만들기"
  4. 위치는 asia-southeast1 선택 (싱가포르)
  5. 보안 규칙은 "테스트 모드로 시작" 선택
  6. 왼쪽 상단 톱니바퀴 ⚙️ → "프로젝트 설정" → 아래로 스크롤 → "앱 추가" → 웹 아이콘 </>
  7. 앱 닉네임 입력 후 등록 → 나오는 const firebaseConfig = {...} 블록 전체를 위에 붙여넣기
